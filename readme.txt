Работа выполнена на ЯП python3 в среде jupiter notebook
программа требует для работы пакет pymorphy. Установить: pip install pymorphy2
Анализируемый текст должен быть представлен в текстовом файле в utf-8 кодировке. Файл должен находиться в папке со скриптом
По умолчанию, имя файла text.txt. Имя файла можно задать в переменной FILE_NAME
В данной версии, алгоритм имеет большое число ложно-положительных срабатываний по причине малых словарей стоп-слов
В последующих версиях планируется расширить словари стоп-слов за счет добавления категорий (например: журналистика)
